## Rocketpay API

Iniciando um Phoenix server:

- Instalar dependências com `mix deps.get`
- Criação de DB com `mix ecto.setup`
- Start Phoenix com `mix phx.server`

Acesso em [`localhost:4000`](http://localhost:4000).
